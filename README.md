<h2 align="center">рџ“‚ ModularizaГ§ГЈo SCM (Service, Module & Controller)<br>

### ![Controller](https://img.shields.io/badge/Controller-TypeScript-white?style=flat-square&logo=typescript) `.controller.ts`
 
Camada de **entrada da API**. Recebe as requisiГ§Гµes HTTP e define as rotas (`@Get`, `@Post`, `@Put`, `@Delete`). NГЈo contГ©m lГіgica de negГіcio вЂ” apenas delega ao Service. Г‰ aqui que os decorators do Swagger (`@ApiOperation`, `@ApiResponse`) sГЈo aplicados.
 
**Arquivos neste projeto:** `auth.controller.ts` `users.controller.ts` `app.controller.ts`
 
---
 
### ![Service](https://img.shields.io/badge/Service-TypeScript-white?style=flat-square&logo=typescript) `.service.ts`
 
Camada de **lГіgica de negГіcio**. Processa os dados recebidos do Controller, aplica as regras da aplicaГ§ГЈo (validaГ§Гµes, hash de senha com `bcrypt`, geraГ§ГЈo de JWT) e comunica com o banco via Prisma. Injetado no Controller via `@Injectable()`.
 
**Arquivos neste projeto:** `auth.service.ts` `users.service.ts` `prisma.service.ts`
 
---
 
### ![Module](https://img.shields.io/badge/Module-TypeScript-white?style=flat-square&logo=typescript) `.module.ts`
 
**Unidade de organizaГ§ГЈo** do NestJS. Agrupa e registra o Controller e o Service de um domГ­nio (`imports`, `providers`, `controllers`, `exports`). Permite que outros mГіdulos reutilizem os providers via `exports`. O `AppModule` Г© o mГіdulo raiz que importa todos os demais.

<img src="https://img.shields.io/badge/Password_Hash-111827?style=flat-square&logo=letsencrypt&logoColor=purple"/>
</h2>
 
O `bcrypt` converte a senha em texto puro em um **hash irreversГ­vel** antes de salvar no banco. O salt rounds `10` define o custo computacional вЂ” quanto maior, mais seguro e mais lento.
 
```ts
import * as bcrypt from 'bcrypt';
 
// No cadastro (users.service.ts / auth.service.ts)
const SALT_ROUNDS = 10;
const hashedPassword = await bcrypt.hash(plainTextPassword, SALT_ROUNDS);
 
await this.prisma.user.create({
  data: {
    email,
    password: hashedPassword, // nunca salva a senha em texto puro
  },
});
 
// No login (auth.service.ts)
const isMatch = await bcrypt.compare(plainTextPassword, user.password);
if (!isMatch) throw new UnauthorizedException('Credenciais invГЎlidas');
```
 
O prefixo `$2b$10$` armazenado no banco Г© decodificГЎvel assim:
 
| Segmento рџ›ЎпёЏ | Valor рџ§Є | Significado рџ”‘ |
|------------|---------|--------------|
| `$2b$` | algoritmo | versГЈo do bcrypt |
| `$10$` | cost factor | 10 salt rounds |
| restante | hash | 53 chars = salt + digest |
